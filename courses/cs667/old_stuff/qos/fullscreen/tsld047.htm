The ATM Approach

  • ATM is connection oriented, setting up Virtual Channels at connect time

  • Route Selection is done at Connection setup time using PNNI

  • ATM is a circuit switched cell basic mechanism

  • Signalling is used to indicate the QoS parameters for this connection. Connection admission Control, or CAC is done to decide the fate of this connection at this point
