Mbone and RSVP

  • The Multicast Backbone (Mbone) is a bunch of nodes talking IP Multicast on the Internet

  • Uses Class D of addresses (224.0.0.0 to 239.255.255.255) as group addresses

  • supports groups, Join/Leave using IGMP

  • Router (m-router) takes care of replication, forwarding, reception, scoping, pruning and IGMP based control

  • Delivery is Best Effort
