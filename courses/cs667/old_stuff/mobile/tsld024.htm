Research projects on mobile computing

  • Some of the most promising and exciting areas of research in the field of Mobile computing are:

  • Virtual Network Computing .

    • This project aims to give users access to their home environments from anywhere in the world.

  • Coda and Odyssey projects.

    • Adapting applications and making them mobile aware

  • Monarch project

    • Allowing mobility for machines using the existing IP protocol.
