    Object Oriented
    • Passes full objects as arguments and return values
    • Can pass a Java hashtable for example
    • RPC requires decomposition of hashtable into data primitive, shipping the data to its destination, and finally rebuilding the hashtable.

    Mobile Behavior
    • Can move behavior (class implementations) from client to server and server to client.
    • Modifying the system may require only adding a new object at the server site.
