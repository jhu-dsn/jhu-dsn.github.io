    An example
    • A client wants to perform an operation on an object
    • The object implementation is sitting on a remote server that was not implemented in the same language
    • ORB acts as the translator between the two
    • It finds the object implementation requested by the client
    • It prepares the object implementation to accept the request
    • It communicates the data making up the request
