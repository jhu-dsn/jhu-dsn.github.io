Java 1.1 - Web Designer’s Perspective

  • Now can bundle classes, data, etc. into a JAR (Java Archive) file.

  • Advantages:

    • Single file for simplicity
    • Single connection + compression for efficiency

  • Also supports digital signatures for authentication

  • New Netscape plug-ins distributed as JAR files : Navigator contains a “JAR manager”.
