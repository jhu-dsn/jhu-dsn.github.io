The Web Protocols

HTTP/1.1

  • Multiple items over one cached connection
  • Ageing of files is possible
  • More control of how (or if) an item is cached
  • More complex "give me this language" semantics possible
  • Host specification added - no more wasted IP addresses
  • MD5 integrity checks
  • Protocol upgrading on open connection
RFC-2068: HTTP/1.1
