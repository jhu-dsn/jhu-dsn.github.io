Selecting the server using DNS

  • The shared IP address is listed as the authoritative DNS server for the record (www.domain.net)

  • Each DNS record has several IP addresses.

  • Each DNS server responds with the unique IP address of the server that is closest to it.

    • This can be modified on the fly with DNS dynamic update based on server load, network problems, etc.
