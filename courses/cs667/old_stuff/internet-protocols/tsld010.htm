Inter-AS Routing (AS path)

  • The AS path length is used when deciding which route to use (if there are multiple BGP announcements for a given route).

    • BGP calculates how many ASes a route has traveled through, and picks the route that has the shortest AS path.
    • If this does not provide the desired behavior, then the network administrator has the option of using other BGP metrics to change this behavior.
