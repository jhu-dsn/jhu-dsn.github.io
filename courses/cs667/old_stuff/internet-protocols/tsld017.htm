Intra-AS Routing (OSPF)

  • As you can imagine, this graph can get very large very quickly, and thus can consume a whole lot of CPU time.

    • There are various ways that network engineers ‘cheat’ to make the graph itself smaller.
    • The only networks that absolutely need to be in the graph are those that are also speaking OSPF. As such, any route that is propagated with OSPF can just be listed as attached to a router that is speaking OSPF.
