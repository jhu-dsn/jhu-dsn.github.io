Intra-AS Routing (Protocols)

  • The most popular IGP is called Open Shortest Path First (OSPF).

    • This protocol is based on the Dijkstra SPF algorithm, and is known as a link state protocol.
    • In a nut shell, each router creates a Dijkstra graph of the entire AS, with itself as the root node. It then runs SPF on each node in the network to find the best path to a given destination within the network.
