Design
The DNS has one entry for the web server in question, when the client attempts to resolve this name to an IP address the DNS returns the actual IP address of the web server. Then the client and server communicate as would any typical client and server.
Single Server Model
Advantages:
  • Easy configuration.
  • Inexpensive configuration.

    • Disadvantages:
  • Limited bandwidth.
  • Limited computational resources.
  • No redundancy.

    • Comments:

    Due to its simplicity, this model reigns the Internet, though high traffic sites must use alternate models to provide for their demand, most small servers use this model.
