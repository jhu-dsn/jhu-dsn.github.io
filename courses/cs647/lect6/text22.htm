WEP – description

  • To send a message M from A to B

    • compute integrity checksum (CRC32): c(M) and plaintext P = {M, c(M)}
    • encrypt P using RC4:
    • ciphertext C = P Å RC4(v,k)
    • where key k and initialization vector (IV) v are
    • used to generate keystream RC4(v,k)
    • Transmit C’ = v, (P Å RC4(v,k))

  • To decipher an encrypted message C’, the encryption process is reversed
