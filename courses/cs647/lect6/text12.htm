Wired Equivalent Privacy

  • Security goals: protect link-level transmission

    • Confidentiality
    • Access control
    • Data integrity

  • How?

    • Share a key between all devices
    • Encrypt all packets using the shared key
    • Use a checksum for packet integrity (prevents also an attacker from introducing spoofed traffic

  • Security relies on the difficulty of discovering the secret key through a brute-force attack
