Encryption algorithms

  • Block ciphers: DES,Blowfish, AES

    • transforms a fixed-length block of plaintext (unencrypted text) data into a block of ciphertext (encrypted text) data of the same length.
    • can be used to encrypt a message of arbitrary length, by using techniques known as modes of operation for the block cipher: Example: ECB, CBC.

  • Stream ciphers: RC4

    • generates a keystream and then XORed with the plaintext to obtain the ciphertext
    • can be designed to be exceptionally fast, much faster than any block cipher.
