Watchdog

  • Maintains a buffer of recently sent packets

  • Each overheard packet is compared with the packet in the buffer:

    • In case of a match, the packet in the buffer in removed
    • If the packet, has remained in the buffer
    • for longer than a certain timeout:
    • reliability associated with the node
    • goes down
    • If the number of failures exceeds the
    • threshold value, it determines that the node
    • is misbehaving
