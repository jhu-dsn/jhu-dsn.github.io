Detecting Routing Behavior

  • Main idea: using promiscuous mode of cards

  • Usually network cards are configured to listen to only packets directed to them. It is possible to configure them to listen to all packets circulating into the LAN: Promiscuous Mode

  • Extra facilities added to detect and mitigate routing behavior (extensions to DSR)

    • Watchdog: identifies misbehaving nodes by overhearing transmissions
    • Pathrater: avoids routing packets through these nodes
