Review Routing Protocols

  • Proactive protocols:

    • Constantly update all routing tables as links change.
    • Requires a lot of updates for destinations and paths that may never get used.
    • Examples: Link-state and distance vector routing

  • On-Demand protocols

    • A path is established only when packets need to be routed to that destination
    • Request/Response, routes are cached
