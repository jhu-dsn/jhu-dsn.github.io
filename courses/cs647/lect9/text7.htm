On-Demand Routing

  • Routes are maintained only between nodes which need to communicate (Examples: AODV, DSR)

  • Route discovery: flood, then form one path back from the destination.

  • The path can be selected based on:

    • An incremented path count (AODV)
    • Shortest recorded route during the flood (DSR).

  • Source routes can be included in the packet headers (DSR) or route tables can be maintained by nodes (AODV)


Notes:

resulting large headers can sometimes degrade performance
