How is OLE Different?

  • Some syntax differences, uses UUID’s

  • Mechanism -- fast library linking

    • Shared library linking
    • No need for marshalling/serializing data formats
    • No overhead of communications

  • Includes embedding specification

    • Excel Spreadsheet inside Word Document
    • Specifies WHAT KIND of object can be used, not just how to distribute ANY object
