CS 337/437 Page opens (October 6, 1999)
This page will have information about the CS337/437 Distributed Systems class held during Fall of 1999.
Class Information

Taught by Yair Amir and Jonathan Stanton.

The course teaches how to design and implement efficient tools, protocols and systems in a distributed environment. The course tries to give some hands-on experience as well as some theoretical background. The programming assignments and the project in the course can be done in either C/C++ or Java. These assignments will use a cluster of workstations as well as several software research tools.

Prerequisites:
  • Computer Systems Fundamentals( 600.333 )
  • Data Structures ( 600.226 )

Final Project

Final class project--3 choices

Due Monday December 13, 1999 or January xxxx, 2000.
The three options are:
  1. Build a replicated, distributed Mail Server that can provide mail messages and keep track of the state of messages even when partitions and crashes occur. The assignment is available at this web site and in the ~cs437 directory on the undergrad machines.
  2. Explore high-performance computing on a cluster of workstations using the MOSIX distributed operating system. This will involve writing a program that computes a matrix multiplication in a parallel way and a second program that plays the game of life and utilizes the most possible resources out of a cluster of machines. Talk to Yair for more information.
  3. Any other idea you come up with that fits in with the course. Come talk to us, we are always interested in new ideas.
