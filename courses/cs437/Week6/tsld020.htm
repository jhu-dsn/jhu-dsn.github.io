The protocol (Ben-Or)

    Iteration=0; x = initial value
    Do Forever:
    Iteration = Iteration + 1
    Step 1
    Step 2
    Step 1:
    Broadcast Proposal(Iteration,x)
    wait for n-f messages of type Proposal(Iteration,*)
    if at least n-2f messages have the same value v
    then x = v (that value)
    else x = undefined
