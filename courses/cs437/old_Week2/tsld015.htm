    Message Omissions A Randomized Algorithm

    At initialization one specific process, p, chooses
    a key at random, uniformly from the range [ 1..r ].

    At each round the processes send the following:

  • Initial value.
  • key (for process p only).
  • color

    Each process holds a variable color initialized to
    green. If red message was received, or a message
    was missed, the process sets color to red.
