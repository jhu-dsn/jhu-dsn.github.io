Historical Perspective--Linda

  • Original model of virtual spaces of tuples

  • Created in early 1980’s

  • Added transactions, queries, fault-tolerance in early/mid 1990’s

  • Java adds:

    • strongly typed fields in tuples
    • objects--not data stored in spaces
    • match object type as well as fields
    • leases
    • atomic actions across multiple spaces
