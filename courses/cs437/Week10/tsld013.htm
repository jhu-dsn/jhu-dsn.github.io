Chord

  • Developed at MIT.

  • Main idea: forming a massive virtual ring where every node is responsible for a portion of the periphery.

  • Node IDs and data keys are hashed using the same function into a non-negative space.

  • Each node is responsible for all the (key,value) pairs for which the hash result is less or equal to the node ID hash result, but greater then then next smaller hashed node ID.
