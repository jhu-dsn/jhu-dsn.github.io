Reference Material
This page contains useful web links, and other documentation and reference material related to CS 337/437. Some of this material is provided by other web sites and may change, so please tell us if you notice a problem with these links or they do not seem useful to you.
General Distributed Systems
Building Secure and Reliable Network Applications
Kenneth P. Birman 1996
http://www.manning.com/Birman/index.html 		This is a recommended book for many of the topics covered in lecture. It is currently out of print so it is difficult to purchase a copy. If you are interested, I believe the publisher still has copies available for purchase at the books website.
Network Programming
Unix Network Programming: Volume 1 Sockets and XTI
Richard Stevens 1998 2nd Edition
http://www.kohala.com/start/unpv12e.html 		This is the Best sockets programming book out there. For both learning and reference it is excellent. The website above is the site for the book and has information like the table of contents, errata, a sample chapter, source code, etc.
http://homepages.stayfree.co.uk/zed/net/ Network programming tutorials.
http://www.stardust.com/wsresource/wsresrce.html A WinSock programming source. Winsock is very similar to standard BSD sockets and might be more familiar to windows programmers. HOWEVER, since all C programs must run on the solaris machines you DO NOT want to actually program using this.
http://www.ecst.csuchico.edu/~beej/guide/net/ A guide to network programming. This is very friendly to beginners.
http://www.faqs.org/faqs/unix-faq/socket/index.html This is a FAQ about Unix sockets (the week1 tutorial).
C programming and Unix library functions
http://www.cs.utah.edu/dept/old/texinfo/glibc-manual-0.02/library_toc.html The C library documentation in HTML format.
