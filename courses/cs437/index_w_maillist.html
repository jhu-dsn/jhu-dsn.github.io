Class Information

Teaching Assistant

The course teaches how to design and implement efficient tools, protocols and systems in a distributed environment. The course gives some hands-on experience as well as some theoretical background. The programming assignments and the project in the course should be done in C / C++ with some exceptions. User interface parts of programs can be built in java if that is desired. The assignments will be evaluated on the Ugrad1-Ugrad7 cluster in the undergrad lab.

For more information about the course please contact the instructor.
Prerequisites:
