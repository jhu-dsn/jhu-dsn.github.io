Security Implementation in Spread

  • Authentication and access control:
    • General framework supporting many different authentication methods.
    • IP, password.
    • Any application access control policy.
  • Encryption and key management done at the library level (Secure Spread):
    • Encryption: Blowfish.
    • Key management: supports five different key agreement protocols, among them a robust protocol based on GDH.
