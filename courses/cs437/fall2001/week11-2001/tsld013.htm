Group Key Management (cont.)

  • Two aspects:
    • Computing the key
    • Distributing the key.
  • Methods:
    • Fixed Key Server: one fixed server selects
      and distributed the key.
      IT DOES NOT SUPPORT NETWORK
      PARTITIONS !
    • One member of the group acts as a key server. These methods try to minimize the distribution cost.
    • Contributory methods: each member contributes a share to the group key.
