    Message Omissions A Randomized Algorithm (cont.)

    Decision Rule:

    A process decides 1 after r rounds if it knows that
    at least one process started with 1, it knows the
    value of key, and it has received all the messages
    in all the first key rounds and all of them were
    green. Otherwise, it decides 0.
