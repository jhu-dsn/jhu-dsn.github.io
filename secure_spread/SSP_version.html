NAME

SSP_version - Return the version of the Secure Spread (ssplib) library  

SYNOPSIS

#include <ssp.h>
int SSP_version( int *major_version, int *minor_version, int *patch_version);  

DESCRIPTION

SSP_version returns the version of the Secure Spread (ssplib) library.

The major_version is the most significant number of the version.

The minor_version is the second number of the version.

The patch_version is the least significant number of the version.

 

RETURN VALUES

Returns 1 on success or 0 if one of the supplied parameters is NULL.

 

BUGS

None.
