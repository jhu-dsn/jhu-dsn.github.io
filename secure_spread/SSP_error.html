NAME

SSP_error - Secure Spread error string reporting  

SYNOPSIS

#include <ssp.h>
void SSP_error(int error);  

DESCRIPTION

SSP_error prints an error string based on the error number it was given.  

RETURN VALUES

Returns nothing.  

BUGS

None.
