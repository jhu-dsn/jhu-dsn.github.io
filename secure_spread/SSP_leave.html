NAME

SSP_leave - leave a group  

SYNOPSIS

#include <ssp.h>
int SSP_leave(mailbox mbox, const char *group);  

DESCRIPTION

SSP_leave leaves a group with the name passed as the string group. If the group does not exist amoung the Spread daemons this operation is ignored, otherwise the client leaves the existing group.

The mbox of the connection upon which to leave a group is the first parameter.

The group
 string represents the name of the group to leave.

 

RETURN VALUES

Returns 0 on success or one of the following errors ( < 0 ):
ILLEGAL_GROUP
The group given to leave was illegal for some reason. Usually because it was of length 0 or length > MAX_GROUP_NAME
ILLEGAL_SESSION
The session specified by mbox is illegal. Usually because it is not active.
CONNECTION_CLOSED
During communication errors occured and the leave could not be initiated.
 

BUGS

None.
