Securing Peer Groups

  • Much more difficult than two-party security

  • Group properties:

    • Membership changes over time.
    • State of group is complex.
    • Many parties can cause asynchronous events.
    • Trust involves more than n2 relationships.

  • First step is to securely create a shared secret. We can then use it to encrypt or sign messages, and provide other security services.
