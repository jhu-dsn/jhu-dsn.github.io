Why Secure Group Communication?

  • Security goes hand in hand with high availability and fault tolerance.

    • The group communication paradigm worked well for providing these, so it could work well for multicast security.

  • Group communication now supports wide-area groups. In open, wide-area networks like the Internet, security is very important.

  • Feasible without requiring changes to network level or operating system level protocols.
