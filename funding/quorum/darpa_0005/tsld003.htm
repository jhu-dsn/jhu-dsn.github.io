Project Impact - Systems

  • Applied the cost-benefit framework to traditional cluster computing (PVM and Mosix), demonstrated considerable performance gains.

  • Built a fairly popular open-source load balancing module for the Apache web server based on the framework (Backhand).

  • Two Internet-wide metacomputing prototypes (Java Market and Share). Share might become useful in the context of the Sloan telescope mapping project.

  • A PVM-like system for Sun’s Jini, with the Cost-Benefit framework is about to be released.
