From Theory to Practice

  • The “maximum CPU utilization” on a machine is adjusted if a real time job runs on that machine.

  • The Real-Time EPVM, the Real-time Differential PVM, and the Real-Time PVM strategies only consider machines where they can safely place a job.

  • If no machine is safe, it rejects the job.

  • Alternatively it could reject jobs based on benefit.

  • Limitations: only static strategies were considered.
