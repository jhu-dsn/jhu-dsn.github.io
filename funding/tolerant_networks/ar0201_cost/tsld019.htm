Archipelago

  • What does it do?

    • Software that creates a fully functional ad-hoc network of wireless 802.11 devices.
    • Allows Internet applications to run seamlessly on top of that network.
    • Provides infrastructure for wireless peer-to-peer applications.

  • The Archipelago Project serves as a test bed and infrastructure for:

    • Distributed routing algorithms
    • Transport protocols
    • Peer-to-peer applications
