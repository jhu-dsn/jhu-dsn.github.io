Network Level Resiliency

  • A scale of resilient routing protocols:

    • Best path (what we do today).
    • Weighted randomized path - winner picking (new).
    • Gravitational flow (new).
    • Flooding.

  • Each protocol (parameter) is associated with a cost based on current resource utilization. Each application stream is associated with application benefit.

  • Unorthodox thinking. Examples:

    • Buffers in “routers” can be used to provide hop by hop reliability - it is just a question of cost and benefit.
    • Benefit optimization versus fairness.
