Cost-Benefit based Flow Control

  • Utilization-based costs:

    • Based on link utilization.
    • Based on buffer utilization in overlay routers (what we currently use).

  • Benefit associated with packets

    • Packets initiated to the system by sources
    • Sources receive $$$ stream (salary).

  • Direct control on each user’s input stream.

    • Back pressure for “costly” users.

  • Near-optimal (competitive) performance (number of packets put on the network).
