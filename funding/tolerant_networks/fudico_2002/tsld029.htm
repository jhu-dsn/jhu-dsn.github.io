Spines Security Architecture

  • Clients obtain a certificate from a Certificate Authority (CA) (off-line).

    • The CA should be a popular CA system, (we had additional options, but Werner’s talk was convincing).

  • Overlay specific control messages use digital signatures. Client data over secure channel use HMAC.

  • Client authentication and access to the overlay network done through a near-by overlay router.

    • Latency is the predominant factor.

  • Client data encryption and integrity done end-to-end.

    • Unless hardware support exists.
