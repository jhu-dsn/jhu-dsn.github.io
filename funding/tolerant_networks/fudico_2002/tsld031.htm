Achieving Transparency Through Interceptors

    Can be done at various levels. For example:

  • At the operating system level.

    • Intercepting on the same computer.
    • Very operating system dependent.
    • Requires running a privileged process on user computers.

  • At the router level.

    • Messages are routed through an overlay node where they are intercepted.
    • Requires overlay-router computer in the middle.
