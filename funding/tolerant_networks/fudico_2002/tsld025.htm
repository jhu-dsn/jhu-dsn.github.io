Spines Security Building Blocks

  • Use current Public Key Infrastructure (PKI) infrastructure and standards: X509 certificates.

  • Secure channels established using Transport Layer Security (TLS) implementation by the OpenSSL project.

  • Data authentication:

    • HMAC-SHA1 if secure channels are in place.
    • 1024-bit RSA digital signatures if no secure channel is in place (RSA signing is pretty expensive, but verification is fast).
