Wackamole: Dynamic Cluster with N-way Fail-over

  • In contrast to available gateway solutions (Alteon, BigIP, Cisco’s Local-director).
  • Exploiting Spread’s membership service to build a distributed agreement protocol that strictly distributes the cluster’s public ip addresses between the currently functioning servers.
  • Should be useful for locally replicated ftp servers, web servers, possibly DNS servers, etc.
